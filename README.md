MERN Task Tracker

A full-stack Task Tracker application built using the MERN Stack.

Features

- Create Tasks
- View Tasks
- Update Tasks
- Delete Tasks
- Search Tasks
- Filter by Status
- Mark Tasks as Completed
- Toast Notifications
- Responsive Bootstrap UI

Tech Stack

Frontend

- React.js
- Vite
- Bootstrap
- Axios
- React Hot Toast

Backend

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

Installation

Backend

cd server
npm install
npm run dev

Frontend

cd client
npm install
npm run dev

Environment Variables

Create a ".env" file inside the "server" folder.

PORT=5000
MONGO_URI=YOUR_MONGODB_URI
